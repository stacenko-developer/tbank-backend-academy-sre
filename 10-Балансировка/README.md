# Домашнее задание №10 "Балансировка"

Для отработки навыков настройки балансировки трафика Вам необходимо реализовать балансировку Sticky Session на основе HTTP-заголовка (headers).

* Для практики используйте образ из предыдущего домашнего задания: containersol/k8s-deployment-strategies.
* Разверните приложение направьте на него трафик при помощи curl или иным способом (учтите, что запросы в трафике должны содержать заголовок для балансировки)
* Документация по алгоритмам балансировки в Nginx Ingress: https://kubernetes.github.io/ingress-nginx/user-guide/nginx-configuration/configmap/#load-balance

В качестве решения приложите скринкаст и манифесты, использованные для развёртывания. В скринкасте необходимо показать, что разные запросы гарантировано прилипают к своим подам.
