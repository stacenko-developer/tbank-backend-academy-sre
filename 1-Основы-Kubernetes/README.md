# Домашнее задание №1 "Основы Kubernetes"

**В рамках задачи необходимо:**
* Подключиться к k8s по выданной ранее инструкции.
* Ознакомиться с информацией в README в приложенных материалах к семинару (manifests.zip) и выполнить развёртывание нагрузок-примеров по указанным инструкциям.

В качестве решения приложите видеозапись или детальные скриншоты экрана, на которых демонстрируется процесс выполнения инструкций из README для каждого типа ресурса Kubernetes в примерах (Pod, ConfigMap, Deployment и т.д.)
