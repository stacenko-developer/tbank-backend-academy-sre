# Домашнее задание №6 "SLA-SLO-SLI"

## 2 задание

В рамках данного задания Вам необходимо придумать метрики и сформировать SLI, SLO, SLA для измерения доступности приложения OnCall. В качестве примера:
* 95% запросов на добавление дежурства выполняются менее, чем 50 миллисекунд
* 98% запросов к API создания пользователей завершаются успешно
* ...

В качестве ответа опишите минимум две технические или бизнес-метрики необходимые для SLI, и сформулированные показатели надёжности (SLI, SLO, SLA).

## 3 задание

Продолжим работу над измерением доступности. В рамках данной задачи Вам необходимо реализовать мониторинг доступности на базе сформулированных в прошлой задаче метрик и индикаторов. Примерный план действий:
* реализуйте и разверните пробер для проверки выполнения индикаторов
* настройте экспорт метрик пробера в Sage
* реализуйте и разверните приложение для подсчёта SLA на основе данных пробера
* продемонстрируйте полученные данные

**hint:** Примеры выполнения поисковых запросов Sage через API доступны в документации.

В качестве ответа приложите ZIP-архив с исходным кодом реализованных приложений / скриптов, использованными для развёртывания манифестами, короткой демонстрацией с объяснением реализованной схемы мониторинга доступности и результатами подсчёта SLA.

**note:** Артефакты данной практики понадобятся для выполнения следующего домашнего задания (визуализация SLA).
