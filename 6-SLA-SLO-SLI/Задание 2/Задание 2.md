# Выбор метрик и формирование SLI, SLO, SLA

## Метрика №1 - Доступность API удаления команды

Данная метрика показывает процент успешных HTTP-запросов удаления команды. Успешными запросами считаются те запросы, которые возвращают код 2xx после выполнения удаления команды.

### Формирование SLI, SLO, SLA

- **SLI** (Service Level Indicator) - Доля успешных HTTP-запросов удаления команды, выраженная в процентах. Вычисляется по следующей формуле: `(количество успешных запросов / количество всех запросов) * 100`
- **SLO** (Service Level Objective) - 99% запросов к API удаления команды завершаются успешно
- **SLA** (Service Level Agreement) - 99% запросов к API удаления команды завершаются успешно

## Метрика №2 - Время выполнения запроса по созданию команды

Данная метрика показывает время выполнения HTTP-запросов по созданию команды. Время выполнения запроса измеряется в миллисекундах.

### Формирование SLI, SLO, SLA

- **SLI** (Service Level Indicator) - 98-й перцентиль времени выполнения запроса на создание команды
- **SLO** (Service Level Objective) - 98% запросов на создание команды выполняются менее, чем 40 миллисекунд
- **SLA** (Service Level Agreement) - 98% запросов на создание команды выполняются менее, чем 40 миллисекунд
