# Домашнее задание №8 "Alerts"

**Цель:**

Настроить систему оповещений (алертов) с использованием **Sage**, чтобы при нарушении ключевых метрик автоматически отправлялись уведомления в Telegram. Задание выполнять в https://sage.sre-ab.ru/

**Задание:**

1. Настроить адрес для отправки уведомлений (документация):
   * адрес создавайте для своей группы
   * адресом должен быть телеграмм канал
2. Создание алертов (документация):
   * Продумать, какие алерты нужны для мониторинга сервиса. Написать 1-2 алерта на системные метрики и 1-2 алерта на метрики пробера.
   * Один из алертов должен быть написан на js (документация)
   * Показать, как сработал алерт.
